A ferramenta contém 4 funções princiapis:

Nuvem: cria uma nuvem de palavras com os tweets mais recentes de um usuario. Tambm mostra informações basicas do perfil.

Tweets: coloca os Tweets com dados de geolocalizacao de um usuario em um mapa.

Hashtags: coloca os tweets com dados de geolocalizacao em um mapa onde tais hashtags estao sendo utilizadas.

Trends Map: mostra os Trends nas capitais do país onde o recurso esta disponível

Para utilizar é necessário obter chaves para a API do Twitter
